附录: 指标清单
================================================================================

  基础:  Agent Acc / Step Acc / Joint Acc
  排名:  Top-1 Agent Acc / Top-3 Agent Acc
         Top-1 Step Acc / Top-3 Step Acc
         MRR (Agent) / MRR (Step)
  容错:  Step MAE / Step Acc@1 / Step Acc@2
  校准:  ECE (仅 feedback 方法)
  效率:  平均 LLM 调用次数 / 平均输入 Token / 平均输出 Token / 平均延迟
